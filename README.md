## Run locally

Clone the project:
```bash
git clone https://github.com/eduardovsousa/onebitflix_back-end_main
```

Access the repository:
```bash
cd onebitflix_back-end_main
```

Install dependencies:
```bash
npm install
```

Run the server:
```bash
npm run dev
```
